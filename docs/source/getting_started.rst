Getting started
=========================

Setting up chempleter on your device
-----------------------------------------

You can install Chempleter on your device using uv or pip.

Prequisites
^^^^^^^^^^^^^^

* Python>=3.12
* uv (optional but recommended)

See installation instructions.

You can also run the GUI directly using uv:



Using Chempleter's GUI
----------------------

After installing, you can start the GUI using the following commands depending on how you installed chempleter:

* uv
``uv run chempleter-gui``
